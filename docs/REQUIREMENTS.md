## 기능 요구사항

### 상품

#### 상품 목록 조회 (구매자/판매자)

- [x] 상품 목록을 불러올 수 있다.
  - 상품의 이름을 조회할 수 있다.
  - 상품의 가격을 조회할 수 있다.
  - 상품의 이미지 URL을 조회할 수 있다.

#### 상품 추가 (판매자)

- [x] 상품을 추가할 수 있다.
  - 상품의 이름을 추가할 수 있다.
  - 상품의 가격을 추가할 수 있다.
  - 상품의 이미지 URL을 추가할 수 있다.
- [x] 필수 필드 누락 시 400 에러를 응답한다.
  - 가격(goodsPrice)
  - 상품 이름(goodsName)
  - 상품 이미지(goodsImage)
  - 상품 아이디(goodsId)
- [x] 이미 존재하는 상품의 이름과 동일한 상품을 추가한 경우 409 에러를 응답한다.
- [x] 가격(goodsPrice)는 0보다 큰 정수여야 한다
  - [x] 가격이 0보다 낮은 경우 400 에러를 응답한다.
  - [x] 가격이 정수가 아닌 경우 400 에러를 응답한다.
- [x] 상품명은 1 ~ 100자의 문자열이다.
  - [x] 상품명은 1 ~ 100자가 아닌 경우 400 에러를 응답한다.

#### 상품 삭제

- [x] 상품을 삭제할 수 있다.
- [x] 해당 상품이 장바구니에 있으면 함께 제거한다.
- [x] 존재하지 않는 상품을 제거 요청한 경우 404 에러를 응답한다.

### 장바구니

#### 장바구니 상품 목록 조회

- [x] 장바구니 상품 목록을 조회할 수 있다.
  - 전체 금액을 조회할 수 있다.
  - 전체 수량을 조회할 수 있다.
  - 담긴 상품들의 정보를 조회할 수 있다.

#### 장바구니 상품 수량 변경

- [x] 장바구니에 담긴 상품의 수량을 변경할 수 있다.
- [x] quantity는 1 이상 99 이하의 정수여야 한다
  - [x] 조건에 맞지 않는 요청의 경우 400 에러를 응답한다.
  - [x] 존재하지 않는 장바구니 상품의 경우 404 에러를 응답한다.

#### 장바구니에 담긴 상품 제거

- [x] 장바구니에 담긴 상품을 제거할 수 있다.
- [x] 존재하지 않는 장바구니 상품을 제거 요청한 경우 404 에러를 응답한다.

### 기타

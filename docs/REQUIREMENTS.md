# 기능 요구 사항

## 상품

### 조회 기능

- [ ] 상품 목록을 전부 조회할 수 있다.
- [ ] 특정 상품을 조회할 수 있다.

### 등록 기능

- [ ] 상품의 정보(이름, 가격, 이미지)를 입력하여 상품을 등록할 수 있다.

### 삭제 기능

- [ ] 특정 상품을 삭제한다.
- [ ] 상품이 삭제되면, 장바구니에서 제거된 상품이 제거된다.

## 장바구니

### 조회 기능

- [ ] 장바구니에 담긴 상품 목록을 조회할 수 있다.

### 변경 기능

- [ ] 장바구니 상품의 수량을 변경할 수 있다.

## 삭제 기능

- [ ] 장바구니에 담긴 특정 상품을 제거할 수 있다.

# 기능 요구 사항

## Product (상품)

### 상품을 조회 할 수 있다

- [ ] 상품 목록을 전부 조회할 수 있다. 조회 후 성공 코드 200과 상품 리스트를 반환한다.
- [ ] 상품 목록이 전부 비었을때도 200을 반환한다.

- [ ] 상품의 아이디로 상품을 조회할 수 있다.
- [ ] 해당하는 상품을 찾을 수 없을 시 404 Not Found 를 반환한다.
  > 리소스를 찾을 수 없기 때문에 404 에러를 반환하게 했습니다.

### 상품을 추가할 수 있다.

- [ ] 필수 필드 누락 시 400 에러를 응답한다

  > 잘못된 요청 구문을 수행하지 않기 위함입니다.

- [ ] 상품명은 최대 100자이다
- [ ] price는 0보다 큰 숫자여야 한다

```
// 필수 필드
name(상품 이름): string
price(상품 가격): number
```

### 상품을 삭제할 수 있다.

- [ ] 해당 상품이 없는 경우 삭제 할 수 없다.
- [ ] 해당 상품을 삭제한다.
- [ ] 모든 장바구니를 돌며, 해당 상품이 있으면 삭제한다

해당 상품이 장바구니에 있으면 함께 제거한다

## Cart (장바구니)

장바구니에 담긴 상품 목록을 조회할 수 있다.
장바구니 상품의 수량을 변경할 수 있다.
장바구니에 담긴 상품을 제거할 수 있다.
